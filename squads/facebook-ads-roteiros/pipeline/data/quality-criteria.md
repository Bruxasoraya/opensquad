# Quality Criteria — Facebook Ads Video Scripts

Critérios de qualidade para avaliar cada roteiro de vídeo produzido pelo squad.

---

## Scoring (1-10 por critério)

### 1. Hook Power (Peso: Alto)
- [ ] Hook tem no máximo 8 palavras
- [ ] Hook para o scroll — passa no "thumb-stop test"
- [ ] Hook usa gatilho emocional claro (dor, curiosidade, medo, desejo)
- [ ] Hook NÃO começa com cumprimento ou apresentação
- [ ] Hook é diferente dos outros roteiros da mesma categoria
- **Mínimo para aprovação: 7/10**

### 2. Pain Accuracy (Peso: Alto)
- [ ] Espelha uma dor REAL e específica do público-alvo
- [ ] Usa linguagem do público (não jargão espiritual rebuscado)
- [ ] Fala em segunda pessoa ("você sente", "você tenta")
- [ ] Conecta com o pain state correto da categoria
- **Mínimo para aprovação: 7/10**

### 3. Authority & Credibility (Peso: Médio)
- [ ] Inclui pelo menos 1 marcador de credibilidade
- [ ] Credibilidade é tecida naturalmente, não anunciada
- [ ] Apresenta o serviço como solução (ponte), não como produto
- [ ] Consistente com o perfil real da Soraya (6 anos, bruxaria natural)
- **Mínimo para aprovação: 6/10**

### 4. Social Proof & Result (Peso: Alto)
- [ ] Inclui pelo menos 1 resultado concreto de cliente
- [ ] Resultado é específico (não vago como "muitas pessoas melhoraram")
- [ ] NÃO faz garantia absoluta de resultado
- [ ] Usa framing seguro: "clientes relatam", "tem funcionado para", "pode ajudar"
- **Mínimo para aprovação: 7/10**

### 5. CTA Effectiveness (Peso: Alto)
- [ ] CTA contém exatamente UMA ação
- [ ] CTA é específico e direto (imperativo)
- [ ] Inclui elemento de urgência/escassez REAL
- [ ] Direciona para o landing page correto da categoria
- **Mínimo para aprovação: 7/10**

### 6. Meta Compliance (Peso: Crítico — Veto)
- [ ] ZERO uso de palavras proibidas (feitiço, amarração, magia negra, garantido)
- [ ] ZERO garantias absolutas de resultado
- [ ] ZERO prazos garantidos ("em 7 dias ele volta")
- [ ] Framing de "orientação espiritual", não "fortune telling"
- [ ] Linguagem de empoderamento, não de dependência
- **Mínimo para aprovação: 9/10. Abaixo de 7 = VETO automático**

### 7. Brand Voice Alignment (Peso: Médio)
- [ ] Tom empático, pessoal e acessível (como definido no perfil)
- [ ] Linguagem conversacional, sem emojis no roteiro falado
- [ ] Perguntas retóricas engajadoras presentes
- [ ] Soraya é apresentada como pessoa real e próxima
- [ ] Foco na dor emocional e solução espiritual
- **Mínimo para aprovação: 7/10**

### 8. Script Structure (Peso: Médio)
- [ ] Segue o framework S.P.A.R.K. (Stop, Pain, Authority, Result, Kick)
- [ ] Duração estimada: 30-60 segundos (65-160 palavras)
- [ ] Frases curtas (máximo 15 palavras por frase)
- [ ] Soa natural quando lido em voz alta
- [ ] Tem ritmo — alternância entre frases curtas e médias
- **Mínimo para aprovação: 6/10**

### 9. Category Accuracy (Peso: Médio)
- [ ] Roteiro é específico para a categoria (não genérico)
- [ ] Landing page correto mencionado/direcionado
- [ ] Produto/serviço correto descrito
- [ ] Preço/condições corretas (quando mencionados)
- **Mínimo para aprovação: 8/10**

### 10. Differentiation (Peso: Médio)
- [ ] Os 3 roteiros da categoria usam hooks diferentes entre si
- [ ] Os 3 roteiros usam ângulos emocionais diferentes
- [ ] Nenhum roteiro poderia ser usado por um concorrente sem alteração
- [ ] Cada roteiro oferece uma perspectiva única sobre o mesmo serviço
- **Mínimo para aprovação: 6/10**

---

## Decision Rules

| Condição | Veredicto |
|----------|-----------|
| Média geral >= 7/10, nenhum critério abaixo de 4/10 | APROVADO |
| Média geral >= 7/10, critério não-crítico entre 4-6 | APROVAÇÃO CONDICIONAL |
| Média geral < 7/10 | REJEITADO |
| Qualquer critério abaixo de 4/10 | REJEITADO (hard trigger) |
| Meta Compliance abaixo de 7/10 | VETO AUTOMÁTICO |
| 3+ ciclos de revisão com mesmos problemas | ESCALAR para usuário |

---

## Veto Conditions (Rejeição Automática)

Rejeitar e refazer se QUALQUER condição for verdadeira:

1. Roteiro contém palavras proibidas pelo Meta (feitiço, amarração, magia negra, garantido, cura)
2. Roteiro faz garantia absoluta de resultado ("ele vai voltar", "resultado garantido")
3. Roteiro direciona para landing page errado
4. Hook tem mais de 8 palavras ou começa com cumprimento
5. Roteiro não tem CTA ou tem múltiplos CTAs
6. Duração estimada fora de 30-60 segundos
7. Roteiro é genérico demais (poderia servir para qualquer tarólogo)
